## 🔑 KEY CONCEPTS GLOSSARY

### Translation Flow
- **Frontend** = Browser (React, TypeScript)
- **Backend** = Python server (FastAPI)
- **Model** = IndicTrans2 (AI translator)

### Model Components
- **IndicProcessor** = Text preprocessor/postprocessor
- **Tokenizer** = Breaks text into tokens
- **Encoder** = Understands input language
- **Decoder** = Generates output language

### Optimization Techniques
- **Batch Processing** = Translate multiple texts together
- **KV Cache** = Remember past attention (faster)
- **Quantization** = Use fp16 instead of fp32 (faster)
- **Early Stopping** = Stop at first end token (faster)

### Performance Metrics
- **BLEU Score** = Translation quality (34-41 range)
- **nodes/sec** = Translation speed
- **GPU** = Nvidia/AMD graphics card (20-50x faster)
- **CPU** = Fallback option (always available)

## 📞 HELP & SUPPORT

### Common Questions

**Q: How long does translation take?**
A: First time: 15-30 seconds (GPU) or 60-120s (CPU)
   Cached: 0 milliseconds! ⚡

**Q: What languages are supported?**
A: 8 Indian languages: Hindi, Tamil, Telugu, Bengali, Marathi, Gujarati, Kannada, Malayalam

**Q: Does it use internet?**
A: No! Everything runs locally on your machine.

**Q: Can I improve speed?**
A: Get a GPU (10-20x faster) or use cache (instant!)

**Q: Is the translation good quality?**
A: Yes! BLEU scores 34-41 (professional quality)
